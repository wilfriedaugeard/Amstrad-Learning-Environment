################# INSTALLATION OPENAI #################

Main depedendences:
    $ sudo apt-get install libsdl1.2-dev libsdl-gfx1.2-dev libsdl-image1.2-dev cmake

To install the Python module:
    $ pip install .


Compilation:
    $ mkdir build && cd build
    $ cmake -DUSE_SDL=ON -DUSE_RLGLUE=OFF -DBUILD_EXAMPLES=ON ..
    $ make -j 4

Compilation with Rl_glue
    $ cmake -DUSE_SDL=ON -DUSE_RLGLUE=ON -DBUILD_EXAMPLES=ON ..
    $ cd doc/examples/
    $ make -f Makefile.rlglue  #produces an error...


################# USAGE #################

Example of usage without displayed screen with a random agent
    $ ./ale -game_controller fifo -player_agent random_agent ale_py/tests/fixtures/tetris.bin

Example of usage with displayed screen with a single action agent
    $ ./ale -game_controller fifo -player_agent single_action_agent -agent_epsilon 0.66 -agent_action 3 -display_screen true -frame_skip 1 ale_py/tests/fixtures/tetris.bin
